Desvantagens do banco de dados não relacional

Consistência

A consistência eventual pode levar a estados temporários inconsistentes.

Consultas complexas

Pode ser mais difícil realizar consultas complexas ad-hoc.

Maturidade

Algumas soluções NoSQL podem ser menos maduras e com menos suporte do que RDBMS (sistema de gerenciamento de banco de dados relacional).

Limitações do servidor

A escalabilidade vertical tem uma limitação em um único servidor; é necessário criar outro servidor para expandir a capacidade.

Preço

O custo da escalabilidade vertical é mais alto, mas não é necessário investir em estrutura separada; você paga por uma solução integrada.

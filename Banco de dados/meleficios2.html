Desvantagens do banco de dados não relacional

Consistência

A consistencia eventual pode levar a estados temporários inconsistentes.

Consultas complexas

Pode ser mais dificil realizar consultas complexas a ad-hoc.

Maturidade

Algumas soluções NoSQL podem ser menos maduras e com menos suporte do que RDBMS(programa usado para criar, atualizar e gerenciar bancos de dados relacionais)

Limitações do servidor

A escalabilidade vertical ele tem uma limitação de um server ele precisa criar outro server para falar sobre as mesas coisas

Preço

Preço da escalabilidade vertical é mais caro, mas não precisa gastar com a estrutura separada, você paga e ja vem a estrutura
